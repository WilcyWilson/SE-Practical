The receptionist will manage the records based on the diagnosis result.
There is a notifier in the PRS which notifies about the medicines that needs to taken and
the follow up dates.
There is also a recommender which recommends about different excercises and diets.
Doctors will also be notified about the timing and about the daily activities.
